# RocketShoes_Web
  Loja construída com  ReactJs, Redux e Redux Saga.
  Aplicação desenvolvida no <a href="https://rocketseat.com.br/bootcamp" target="_blank"> Rockeseat GoStack Bootcamp</a>     
       
  

# Tecnologias Utilizadas :

* ReactJS
* Redux
* Redux Saga
* Axios
* React-Icons
* React Router
* history
* styled-components
* Reactotron
* ESLint
* immer
* json-sever
* polished
* React Toastify
